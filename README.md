# WhittedRT
This is a Whitted Ray Tracer written in C++.

## Features

* Polygonal Meshes, OBJ files
* Image and Procedural Textures
* Transformations and Instancing
* Multithreading
* Anti-Aliasing
* Bounding Volume Hierarchies
* Scene Parsing
* Zero Dependencies, Pure Modern C++
